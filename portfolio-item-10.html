Papa Philly'sPizzaria!!

Pizza & Topping Management Application

A recent project I made for a coding exercise, simply made to display, test and critique my current Full Stack Development cababilities.

As the subtitle reads, it's a management application of various pizzas and their respective toppings. It allows users, specifically those of either a Chef or Management role, to either see/add/delete/edit toppings and do the same with pizzas using that topping database.

This project is currently incomplete. It's made in Visual Studio Code with HTML5, CSS3, SQL, and JavaScript w/ jQuery and Node.js.
