Where to go? Travel IO

Getting To Grips with Web APIs!

This was one of my first collaborative efforts, made by me and a handful of friends and former classmates.

A simple travel application that prompts a user to search any given city for their respective in-the-moment details, such as weather, hotel vacancies, restuarants and events!

The idea came to us after thinking specifically about user intercontivity on a local level with any given city. This was a chance we gave ourselves to demonstrate our working knowledge of Web API's, applied different ways to retrieve various types of information local to any entered city.

This project was largely put together in visual studio code and github, using HTML, CSS, JavaScript, Web APIs and RESTful APIs. A large part of my contribution was on some of the WEB API and CSS for the landing page.
