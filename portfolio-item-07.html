READMEMAKER

Easily create READEME files.

A project straight out of my time in school, during a time I was getting familiar with Node.js. I still use it to create all my README files for all my projects.

When prompted, the terminal takes the user through a series of questions and user inputs, ultimatley collecting the information into a neatly laid out README file for any given project.

This nifty project was put together with node.js and npm.
